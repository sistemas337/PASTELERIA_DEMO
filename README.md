# Demo de gestión para pastelería

Incluye:
- Dashboard con ventas y pedidos.
- Gestión de pedidos con estados.
- Catálogo de productos.
- Directorio de clientes.
- Inventario con alertas.
- Guardado local mediante localStorage.

## Uso
1. Abre `index.html` en tu navegador.
2. Para publicarlo, sube los tres archivos a un repositorio de GitHub.
3. Activa GitHub Pages desde Settings > Pages.

Este demo no utiliza servidor ni base de datos externa.
